# MyChat - Real-time Android Chat Application

MyChat is a real-time chat application for Android, built using Firebase. It allows users to sign up, log in, and exchange messages instantly in a global chat room.

## Features

- **User Authentication:** Secure sign-up and sign-in using Firebase Authentication (Email/Password).
- **Real-time Messaging:** Messages are sent and received instantly using Firebase Realtime Database.
- **Modern UI:** A clean, intuitive user interface with chat bubbles for sent and received messages.
- **Sign-out Functionality:** Users can securely sign out of their accounts.
- **Cloud-Based:** All chat data is stored securely in the cloud.

## Technologies Used

- **Android (Java):** The core application is built using native Android with Java.
- **Firebase:**
  - **Firebase Authentication:** For managing user accounts.
  - **Firebase Realtime Database:** For storing and syncing chat messages in real-time.
  - **FirebaseUI:** To rapidly build the authentication screens and the database list adapter.
- **Material Components:** For modern UI elements like FloatingActionButton and chat bubble styling.
- **Git & GitHub:** For version control.
